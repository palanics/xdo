# xdo
A simple ToDo application

## Technology used
  - Ruby
    - Sinatra
    - her - ORM
    - gems used
      - `sinatra`
      - `thin`
  - AngularJS
  - Bootstrap and jQuery
  - MongoDB


  ## Installation
    - Install `sinatra` using `gem install sinatra`
    - Run `xdo.rb` using `ruby xdo.rb`
